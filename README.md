# Airrohr-Cli
*Deine Airrohr Sensordaten im Terminal*

Ein Airrohr ist ein [Selbstbau Feinstaubsensor](http://luftdaten.info/feinstaubsensor-bauen/) des OK Lab Stuttgart.
